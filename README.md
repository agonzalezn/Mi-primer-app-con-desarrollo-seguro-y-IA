# Mi-primer-app-con-desarrollo-seguro-y-IA# 
🌤️ Weather App Mock API

Proyecto de aprendizaje: desarrollo seguro con IA en el IES Rafael Alberti.

## Instalación
`npm install`

## Uso
`npm start`
*Servidor en http://localhost:3000*

## Tests
`npm test`

## Cómo se usó la IA
- Handler `GET /weather/:city` generado con IA.
- Revisión manual línea a línea implementando validación estricta de parámetros.
- Tests expandidos con edge cases manuales (ej: bloqueo de números en el nombre de la ciudad).

Ver `RULES.md` para la política completa.
